# Pizza Menu
A small pizza menu application built with Angular 20.

The project was created as part of a front-end assignment and follows the provided design and data structure.
## Features

- Accordion menu cards
- Only one item can be open at a time
- Editable prices
- Disable a size and set its price to 0.00
- Save changes using localStorage
- Undo changes separately for each menu item
- Responsive and keyboard-accessible UI
- No third-party UI or state management libraries

## Technologies
- Angular 20
- TypeScript
- SCSS

### Requirements
- Node.js compatible version
- npm
- Angular CLI 20

## Accessibility
The application uses semantic HTML native form controls ARIA attributes visible focus styles and reduced motion support following WCAG 2.1 AA guidelines
